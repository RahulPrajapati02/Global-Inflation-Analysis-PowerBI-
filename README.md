# Global Inflation Analysis Dashboard

This project is a Power BI dashboard that analyzes global inflation trends across different countries and years.

Tools Used:
Power BI
Excel Dataset

Features:
KPI cards showing highest, average, and lowest inflation
Year-wise inflation trend analysis
Country-wise inflation map visualization
Interactive year filter

Purpose:
This dashboard helps analyze global inflation trends and compare inflation rates across different countries.
